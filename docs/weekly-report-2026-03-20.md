## 核心功能完善（6项）

### 1. 并发安全机制优化
- 明确并发安全支持任意数量设备同时接单，无重复分票
- 实现方式：SQLite 进程锁 + Redis 原子操作 + 条件 UPDATE
- 完成 20 设备并发压力测试验证

### 2. 设备管理增强
- 修复设备限制检查逻辑，过滤过期会话不占用设备名额
- 会话清理改为读取管理员配置的超时时长（不再硬编码 3 小时）
- 接单页显示当前设备名称
- 中签记录关联并显示设备信息

### 3. B 模式资源保留策略
- B 模式始终为 A 模式保留 20 张票作为缓冲
- 防止 B 模式批量接单导致 A 模式无票可接

### 4. 数据导出体验优化
- 今日处理清单下载：空结果返回友好提示而非 404 错误
- 新增未到期票数量统计，前端 toast 提示用户
- 修复导出文件乱码问题
- 上传成功后自动清空队列，失败项保留

### 5. 统计功能增强
- 新增今日各设备出票统计，多设备时自动显示统计表
- 按设备分组展示处理数量

### 6. 基础配置调整
- 基注金额从 2 元调整为 1 元
- 修复 `load_dotenv` 执行顺序问题
