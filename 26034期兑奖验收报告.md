## 一、功能实现清单

### 1. 彩果文件解析 ✓
**文件路径**: `services/result_parser.py`

**实现功能**:
- ✓ 支持带箭头格式的序号解析（如"2→1"）
- ✓ 解析6种玩法的赛果和SP值：SPF、CBF、JQS、BQC、SXP、SF
- ✓ SXP（上下单双）文字转数字映射：
  - 上单 → 0
  - 上双 → 1
  - 下单 → 2
  - 下双 → 3
- ✓ SF（胜负）文字转数字映射：胜→3，负→0，平→1
- ✓ 支持UTF-8和GBK编码自动检测
- ✓ 多次上传以最新为准（upsert逻辑）

**测试结果**: 成功解析25个场次的赛果数据

---

### 2. 中奖计算逻辑 ✓
**文件路径**: `utils/winning_calculator.py`

**实现功能**:
- ✓ 支持6种玩法：SPF、CBF、JQS、BQC、SXP、SF
- ✓ 串关组合计算（笛卡尔积）
- ✓ 延期场次处理：
  - 支持关键词：延期、postponed、delayed、-
  - 延期场次任何选项都算中奖
  - 延期场次SP值固定为1.0
- ✓ 奖金计算公式：SP值之积 × 2元(基注) × 倍投数 × 1.3(系数)
- ✓ 扣税逻辑：
  - 税前金额 ≤ 10000元：不扣税
  - 税前金额 > 10000元：扣税20%
  - 返回：税前金额、税后金额、扣税金额

**测试结果**:
- 单场测试：13/13通过
- 延期场次测试：4/4通过
- 扣税逻辑测试：2/2通过
