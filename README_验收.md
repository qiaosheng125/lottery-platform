## 常见问题

### Q1: 为什么只有3张中奖？
A: 因为26034期的赛果与大部分投注不匹配。中奖需要所有场次都匹配（串关规则）。

### Q2: 为什么没有扣税？
A: 因为所有中奖票的金额都≤10000元。只有>10000元才扣税20%。

### Q3: 如何验证奖金系数1.3？
A: 查看中奖票的计算过程，例如票ID 5：
```
SP值之积 × 2元 × 3倍 × 1.3 = 3,224.15元
```

### Q4: 如何测试延期场次？
A: 运行 `python test_postponed_tax.py`，会模拟延期场次的计算。

### Q5: 如何测试扣税逻辑？
A: 运行 `python test_postponed_tax.py`，会模拟>10000元的中奖金额。
